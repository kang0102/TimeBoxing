【操作指南】
• 手動修改數值：針對量測角度「點擊滑鼠左鍵」即可直接輸入更改，系統會即時更新圖表與均值。
• 排除異常與備註：在量測角度上點擊「滑鼠右鍵」，可選擇排除該數值並輸入【排除原因/備註】。排除後將不計入圖表與計算。滑鼠游標「懸停」於被排除的數字上，即可顯示備註內容。
• 匯出連動 Excel：點擊上方的綠色按鈕，匯出的 Excel 會自帶 `AVERAGE`、`ABS`、`IF` 等原生函式，可以直接在 Excel 內修改數值進行連動運算。

【名詞定義】
• O_Δ / NK_Δ (差值)：單筆測量中，右側與左側的角度絕對差距 ( |R - L| )。
• Off均L / Off均R：單側的歷史累積平均值。
• Off總均 / NK總均：單筆測量的左右平均 `(L+R)/2` 的歷史累積線。反映機台整體的綜合趨勢。
• T-Off_L / T-Off_R (目標差距)：目標角度減去實際量測角度 ( Target - Measured )。
• 標準差 (Standard Deviation)：位於表尾，提供「樣本(n-1)」與「母體(n)」兩種算法切換，反映整體製程數據的離散程度(穩定度)。
